## Installation

Install `suprsend-go` sdk

<CodeGroup>

```shell bash
go get github.com/suprsend/suprsend-go
```

</CodeGroup>

## Initialization

For initializing SDK, you need workspace_key and workspace_secret. You will get both the tokens from your [Suprsend dashboard](https://app.suprsend.com/en/production/developers/api-keys) (Developers -> API Keys).

<CodeGroup>

```go Request
package main

import (
  "log"

  suprsend "github.com/suprsend/suprsend-go"
)

// Initialize SDK
func main() {
  opts := []suprsend.ClientOption{
    // suprsend.WithDebug(true),
  }
  suprClient, err := suprsend.NewClient("__workspace_key__", "__workspace_secret__", opts...)
  if err != nil {
    log.Println(err)
  }
}
```

</CodeGroup>
