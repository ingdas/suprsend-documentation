## Installation

Install `suprsend-go` sdk

<CodeGroup>
  ```shell bash
  go get github.com/suprsend/suprsend-go
  ```
</CodeGroup>

## Initialization

For initializing SDK, you need **API KEY** and **API SECRET**.

<CodeGroup>
  ```go go
  package main

  import (
    "log"

    suprsend "github.com/suprsend/suprsend-go"
  )

  // Initialize SDK
  func main() {
    opts := []suprsend.ClientOption{
      // suprsend.WithDebug(true),
    }
    suprClient, err := suprsend.NewClient("__api_key__", "__api_secret__", opts...)
    if err != nil {
      log.Println(err)
    }
  }
  ```
</CodeGroup>

Replace **API KEY** and **API SECRET** with your workspace values. You will get both the tokens from [Suprsend dashboard](https://app.suprsend.com/) (***Settings page -> "API keys"*** section). For more details, check the [documentation on 'Workspaces'](/docs/workspace).
