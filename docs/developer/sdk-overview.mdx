SuprSend SDKs are lightweight wrappers around the REST APIs, providing out-of-the-box methods that reduce repetitive code and simplify integration.

## Server-side SDKs

Server-side SDKs let you integrate SuprSend in your backend applications and handle notification orchestration with minimal setup.
