## Installation

There are two ways you can install SuprSend SDK into your app:

<Accordion title="Using Cocoapods" defaultOpen={false}>
  Refer complete docs [here](/docs/integrate-ios-sdk#cocoapods-integration).&#x20;

  <Steps>
    <Step title="Add the below pods to your pod file.">
      <CodeGroup>
        ```ruby Podfile
        pod 'SuprsendCore'  
        pod 'SuprSendSdk'
        ```
      </CodeGroup>

      ![](/images/docs/340777d-Screen_Shot_2022-09-26_at_5.09.01_PM.png)
    </Step>

    <Step title="Install pods in the project using the below command in the terminal.">
      <CodeGroup>
        ```ruby Shell
        pod install
        ```
      </CodeGroup>
    </Step>
  </Steps>
</Accordion>

<Accordion title="Manual Installation" defaultOpen={false}>
  Refer to complete docs [here](/docs/integrate-ios-sdk#manual-integration)



  Follow the below steps to do manual integration of SuprSend SDK in the iOS application.

  <Steps>
    <Step title="Download these two zip files">
      [SuprSendSdk.xcframework.zip](https://github.com/suprsend/SuprSend-iOS-XCFramework/releases/download/1.0.0/SuprSendSdk.xcframework.zip) and [SuprsendCore.xcframework.zip](https://github.com/suprsend/SuprSend-iOS-XCFramework/releases/download/1.0.0/SuprsendCore.xcframework.zip) and unzip/ extract them.
    </Step>

    <Step title="Move them to your project folder.">

    </Step>

    <Step title="Select your project target">
      Select your project target and move to the section ***Frameworks, Libraries, and Embedded Content***

      ![](/images/docs/0fbe367-Screenshot_2022-04-08_at_13.45.44.png "Screenshot 2022-04-08 at 13.45.44.png")
    </Step>

    <Step title="Click on the + button --> Add Other --> Add Files">
      ![](/images/docs/85d0b8f-Screenshot_2022-04-08_at_13.47.32.png "Screenshot 2022-04-08 at 13.47.32.png")
    </Step>

    <Step title="Navigate to your project folder">
      After navigating to the project folder, select ***`SuprSendSdk.xcframework`*** and ***`SuprsendCore.xcframework`***
    </Step>
  </Steps>


</Accordion>

## Initialization

<Steps>
  <Step title="Add the below lines of code in your AppDelegate.swift.">
    <CodeGroup>
      ```swift AppDelegate.swift
      import SuprSendSdk 

      // inside application didFinishLaunchingWithOptions method
      let suprSendConfiguration = SuprSendSDKConfiguration(withKey: "<your_workspace_key>", secret:"<your_workspace_secret>")
      SuprSend.shared.configureWith(configuration: suprSendConfiguration, launchOptions: launchOptions)   
      ```
    </CodeGroup>
  </Step>

  <Step title="Set up workspace key and secret">
    Replace ***`<your_workspace_key>`*** and ***`<your_workspace_secret>`*** with your project workspace key and workspace secret, which you can find in the SuprSend dashboard.
  </Step>
</Steps>
