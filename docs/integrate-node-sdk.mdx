## Installation

<CodeGroup>
  ```shell npm
  npm install @suprsend/node-sdk@latest
  ```

  ```shell yarn
  yarn add @suprsend/node-sdk@latest
  ```
</CodeGroup>

## Initialization

<CodeGroup>
  ```javascript javascript
  const {Suprsend} = require("@suprsend/node-sdk");

  const supr_client = new Suprsend("WORKSPACE KEY", "WORKSPACE SECRET");
  ```
</CodeGroup>

Replace `WORKSPACE KEY` and `WORKSPACE SECRET` with your workspace values. You will find them on SuprSend Dashboard -> Settings -> API Keys.
