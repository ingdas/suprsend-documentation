This section is a step-by-step guide to integrate FCM as your service provider for sending Android Push notifications.

## Pre-requisites

Integrate FCM in your application:

[Android Native](/docs/android-firebase-fcm-push-integration)

[React Native](/docs/react-native-firebase-fcm-push-integration)

[Flutter](/docs/flutter-firebase-fcm-push-integration)

## Add FCM config on vendor page

On the SuprSend dashboard, go to vendor page from side panel and click Androidpush -> FCM and add service account key file.

![](/images/docs/b4ca25d-Group_5.png)

### Get Service Account Key File

Add Service Account Key json file. You'll get this file from your Firebase Console Project settings -> "Service accounts" tab

![](/images/docs/e2d76a2-Group_6.png)
