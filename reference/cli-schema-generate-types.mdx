The `generate-types` command creates type definition files in a given language for all enabled SuprSend schemas from a specified workspace.

## Syntax

```bash
suprsend generate-types <language> --output-file <output-file> [flags]
```

**Arguments:**
| Argument | Description |
|----------|-------------|
| `<language>` | The programming language to generate types for. Supported languages - `typescript`, `go`, `python`, `java`, `kotlin`, `swift`, `dart` |
| `--output-file` | Path to the output file where types will be generated (required) |


## Flags

| Flag | Description | Default |
|------|-------------|---------|
| `-h, --help` | Show help for the command | – |
| `--output-file string` | Output file for generated types | – |
| `--build-flags string` | Flags to generate types in a certain way (e.g. `--zod` for TypeScript) | – |
| `--mode string` | Mode of schema to fetch (draft, live) | `live` |
| `--workspace string` | Workspace to get schemas from | `staging` |

## Example

<CodeGroup>

```bash python
# Generate TypeScript types from all enabled schemas in staging workspace
suprsend generate-types python --output-file suprsend-types.py

# Generate Python types with Pydantic in production workspace
suprsend generate-types python --pydantic --workspace production
```

```bash typescript
# Generate TypeScript types from all enabled schemas in staging workspace
suprsend generate-types typescript --output-file suprsend-types.ts

# Generate TypeScript types with Zod in production workspace
suprsend generate-types typescript --zod --workspace production
```

```bash go
# Generate Go types from all enabled schemas in staging workspace
suprsend generate-types go --output-file suprsend-types.go
```

```bash java
# Generate Java types from all enabled schemas in staging workspace

# Generate Java types with Lombok in production workspace
suprsend generate-types java --lombok --workspace production

```

```bash kotlin

```


