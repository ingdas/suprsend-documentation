## Syntax

```bash
suprsend profile use [flags]
```

## Flags

| Flag | Description | Default |
|------|-------------|---------|
| `-h, --help` | Show help for the command | – |
| `--name string` | Profile name to set as active (required) | – |

## Example

```bash
suprsend profile use --name default
```
