## Syntax

```bash
suprsend profile remove [flags]
```

## Flags

| Flag | Description | Default |
|------|-------------|---------|
| `-h, --help` | Show help for the command | – |
| `--name string` | Name of the profile to remove (required) | – |

## Example


```bash
suprsend profile remove --name unused-profile
```
