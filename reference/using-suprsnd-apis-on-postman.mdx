---
title: "SuprSend APIs on Postman"
---

Our [<u>Postman collection</u>](https://www.postman.com/suprsend/workspace/suprsend/collection/27786422-d77a13c1-8f59-406d-9669-078a10d52521) is a good way to get familiar with the SuprSend API. Here's how to get started.

1. Fork our [<u>Postman Collection</u>](https://www.postman.com/suprsend/workspace/suprsend/collection/27786422-d77a13c1-8f59-406d-9669-078a10d52521)
2. Visit `Variables` tab on postman collection
3. From SuprSend's platform get the env variables like authorization headers, workspace key, etc and set it in the variables
4. Navigate through the collection and start using the SuprSend APIs
