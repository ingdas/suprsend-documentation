## Syntax

```bash
suprsend event list [flags]
```

## Flags

| Flag | Description | Default |
|------|-------------|---------|
| `-h, --help` | Show help for the command | â€“ |  
| `-l, --limit int` | Limit the number of events to list | `20` |
| `-f, --offset int` | Offset into the list of events | `0` |
| `-w, --workspace string` | Workspace to list events from | `staging` |

## Example

```bash
# List events in staging workspace (default)
suprsend event list

# List events in production workspace
suprsend event list --workspace production

# List events with limit and offset
suprsend event list --limit 10 --offset 5
```
